Changes since the 1.74.0 release

Note: This document lists changes only since the 1.74.0 release. If you instead want a record of the complete list of changes for the codebase over its entire history, you can obtain one by running the following commands: 

  svn checkout https://docbook.svn.sourceforge.net/svnroot/docbook/trunk/xsl
  svn log --xml --verbose xsl > ChangeHistory.xml
